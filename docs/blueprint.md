# **App Name**: Bit4Code

## Core Features:

- Institutional Page Management: Render essential institutional pages (Home, Services, Contact, Privacy, Terms of Use) with Next.js App Router for high performance and SEO readiness.
- Dynamic Brand Identity: Display a custom, responsive SVG logo and an interactive, visually appealing hero art component directly implemented in React, without external image assets.
- Advanced Security Headers & Configuration: Implement a comprehensive set of security headers (X-Content-Type-Options, X-Frame-Options, Referrer-Policy, Permissions-Policy, Strict-Transport-Security, Content-Security-Policy) within Next.js configuration for a hardened institutional site.
- SEO Foundation & Site Structure: Generate dynamic metadata, `robots.ts`, and `sitemap.ts` files to establish a solid SEO foundation and improve search engine visibility for all public pages.
- Adaptive User Interface: Provide a fully responsive layout with a fixed header and institutional footer, ensuring optimal viewing and interaction across all device sizes and screen orientations.
- Professional Contact Interface: Present clear contact information with direct `mailto` links, offering a professional and easily accessible channel for institutional communication.

## Style Guidelines:

- A vibrant and modern magenta (#D9328A) reflecting innovation and digital dynamism, used for key actions and highlights.
- A deep, sophisticated graphite (#060606) to provide a stark contrast for text and branding, reinforcing the tech-premium and clean aesthetic.
- A subtle, cool-toned violet (#9F4489) used sparingly for nuanced visual depth and complementing the primary magenta within a dark, modern scheme.
- Body and Headline font: 'Inter' (sans-serif) for a modern, clean, and highly readable typeface that is versatile across headlines and extensive body text, aligning with the professional and tech-savvy brand.
- Utilize custom-designed SVG vector graphics for the brand logo and hero sections, featuring abstract geometric forms, clean lines, and dynamic glowing effects to convey technology, precision, and modernity.
- Employ a responsive grid-based layout for organized content presentation, featuring distinct sections for hero, services, and information. A sticky header and institutional footer ensure consistent navigation and branding across all pages.
- Implement subtle micro-interactions for buttons and navigation links, using smooth 160ms ease transitions on hover to enhance user engagement without distracting from the professional content.
